Artofcircuits artobot

ArtoBot is a set of example Arduino codes for 2 wheel drive and 4 wheel drive experimental robots.

Download this project as a .zip file Download this project as a tar.gz file

Welcome to Art of Circuits artobot

ArtoBot is currently designed around Arduino Mega 2560 but other Boards such as Arduino Uno, Arduino Leonardo, Arduino DUE, Arduino Pro mini, Arduino Pro Micro, or Any Arduino Board which full fills the IO requirements can be used (with minor modifications to orignal scheme). ArtoBot uses HC-06 Bluetooth Module. Any Bluetooth capable Android phone can be used to control ArtoBot using a free Remote Control App from google play store.

ArtoBot Application code written for Arduino IDE can be customized to work with other Bluetooth Remote Apps.
